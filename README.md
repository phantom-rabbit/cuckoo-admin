# Cuckoo-admin 介绍

Cuckoo-admin 是cuckoo-pay 后台管理系统

## 目录介绍

```
.
├── Dockerfile          Dockerfile
├── Makefile            Makefile
├── adm.ini             adm配置文件
├── admin.db            sqlite数据库
├── build               二进制构建目标文件夹
├── config.json         配置文件
├── go.mod              go.mod
├── go.sum              go.sum
├── html                前端html文件
├── logs                日志
├── main.go             main.go
├── main_test.go        CI测试
├── pages               页面控制器
├── tables              数据模型
└── uploads             上传文件夹
```

管理员身份运行后，访问：http://127.0.0.1:80/admin
